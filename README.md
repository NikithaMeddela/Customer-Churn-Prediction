# Customer Churn Prediction

## Project Overview

This project predicts customer churn using Machine Learning techniques. The goal is to identify customers who are likely to leave a company's services based on customer demographics, account information, and service usage patterns.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Google Colab

## Project Workflow

1. Data Collection and Loading
2. Data Exploration and Cleaning
3. Label Encoding of Categorical Features
4. Exploratory Data Analysis (EDA)
5. Train-Test Split
6. Random Forest Model Training
7. Model Evaluation
8. Feature Importance Analysis
9. Business Conclusions

## Model Used

- Random Forest Classifier

## Key Findings

- Customers with month-to-month contracts are more likely to churn.
- Higher monthly charges increase churn risk.
- Customers with longer tenure are less likely to leave.
- Contract type and tenure are major factors influencing churn.
